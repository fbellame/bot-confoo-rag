Farid Bellameche va présenter une session sur comment construire un bot sur vos données personnelles.
Pour cela on va utiliser 
 - l'API OpenAI GPT-3.5 comme LLM (Large language model), 
 - Langchain pour la gestion de la vector Database
 - Steamlit pour l'interface graphique du bot.

Cette présentation à lieu dans le cadre de Confoo.ca, un évènement majeur dans le domaine des technologies à Montréal au mois de février 2024.

Farid Bellameche est un passioné d'IA générative, dev/ops et développeur Java et Python.
Depuis la sortie en novembre 2022 de chat GPT, j'ai exploré comment utiliser les API d'OpenAI pour en faire des chatbot augmenté avec le pattern de RAG (Retrieval Augmented Generation).
J'ai aussi appris à entrainer des LLM (fine tuning) sur des tâches spécifiques.
